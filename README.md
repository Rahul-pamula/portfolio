# Rahul Pamula - AI & Backend Engineer Portfolio

A premium, high-performance personal portfolio website built with React, TypeScript, and Tailwind CSS. The site is optimized for speed, mobile responsiveness, and clean layout presentation.

---

## 🎨 Design System & Stack
- **Theme**: Light Premium (Apple/Stripe aesthetic)
- **Primary Background**: `#FFFFFF`
- **Gradient Accent**: Soft Blue (`#4F8CFF` → `#00C6FF`)
- **Interactive Framework**: React 19 + TypeScript + Vite 8
- **Styling**: Tailwind CSS v3
- **Animations**: Framer Motion
- **Icons**: Lucide React

---

## 📂 Project Structure
* `src/components/`: Modular React sections:
  - `Navbar`: Frosted-glass navigation drawer
  - `Hero`: Primary intro headlines & CTAs
  - `About`: Engineering philosophy and skill columns
  - `Projects`: Interactive metrics cards
  - `Skills`: Categorized technical proficiency grid
  - `Experience`: Impact-focused history timeline
  - `Resume`: Embedded PDF viewer and download triggers
  - `Contact`: Email copy utils, portrait image, and social links
* `public/`: PDF assets and static favicon items
* `src/assets/`: Media files (profile portrait)

---

## 🚀 How to Run Locally
1. Clone or open the repository
2. Install packages:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Access the portal at [http://localhost:5173/](http://localhost:5173/)
